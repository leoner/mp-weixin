大点声儿，不费电

2013-01-23

这几天互联网行业的一个话题就是 Github 被封的事情，昨天李开复老师也疾呼：「封锁GitHub毫无道理，只会导致国内程序员和国际脱轨，失去竞争力和前瞻性」。这样的话无疑会引起共鸣的，该条微博被转发数万次。


看起来，大家的声音有了一点效果，今天下午，Github 在部分地区已经可以正常访问了。我们的声音再大一点，还是有人会听到的。 属于我们的东西，我们要自己争取，不能等别人施舍。大点声儿，不费电。


对于这次 GitHub 为何被封，也有人说可能是因为有人在 GitHub Pages 上托管了敏感内容而导致的。


如果上述是事实，这里我想跟大家探讨一个问题：以前 Twitter / Blogger 等产品的被封其实也都是有人不停在上面发布敏感内容进行一定程度的「污染」。国内大概有一千个用户左右(我自己的估计)，这些用户几乎是到哪里哪里就会被封(我们不去讨论他们在那些网站上发布的内容政治倾向性正确与否，正义与否)。大家怎么看这个事情? 支持他们海外「远距离」战斗还是呼吁他们回到国内战场来「近身」搏斗?


你的观点是什么?


今日消息：腾讯移动事业群已确认解散，2月1日将正式发文通告。


大互联网公司是否有必要搞一个单独的无线/移动部门? 我个人的建议是似乎没这个必要。以支付宝为例子，以前是分了拆，拆了分。就是一个折腾。对外产出的产品依然只有一个支付宝客户端，还做出来啥了呢? 没有人能记住一家公司有什么部分，重要的是做出来什么产品。


消息之二：有人说 PPTV 资金链吃紧，取消年终奖。侧面了解了一下，的确有不少员工已经跑出来了。


这个冬天有点冷。PPTV 有一阵子内容还是不错的，最近作为忠实用户的我都已经不怎么用了，更多是用搜狐视频客户端了。

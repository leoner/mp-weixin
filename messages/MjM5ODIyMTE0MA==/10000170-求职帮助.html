--今日问小道--


问: 能不能讲讲大学生投简历，需要注意点什么（投互联网公司），或者你自己喜欢什么样的简历?


答：这是我以前在 JobsDigg.com 上贴过的简历中的加分点。来自我的微博( @Fenng )的内容整理.

现在看工程师的简历，容易加分的亮点是：

1. Github 上的项目地址. (是否具备一定的网络协作能力)
2. Stack Overflow 有积分. (或是否通过英文搜索引擎查询内容而非百度)
3. Twitter ID. (是否知道翻墙这个事儿)

看产品经理的简历，容易加分的亮点是：

1.知乎 ID (可以去查看发过的内容，判断是否有持续学习的能力)
2.长期更新的Blog (良好的写作能力)

注意是说*容易加分*，有些硬性要求当然不能跳过。

李开复的评论：

1）看邮件用什么邮箱（有些加分，有些减分）
2）看他发的微博、关注什么人、什么人关注他
3）搜索（网络搜索+微博搜索）他

再补充发送求职信的建议：

求职者发送简历的时候，邮件标题很重要。最好以{名字}的简历+ {目标职位} 的形式. 比如，「梁朝伟的简历 求职网络编辑」。这样招聘方收到邮件，一目了然。


发送格式最好是正文粘贴一份内容加上 PDF 格式. 最好不要用 Word 格式，当然能不用附件最好。


另外也有人问过毕业生找工作的事情。恰好昨天翻看《编程随想录》，又看到 Joel 写的对计算机专业学生的几条建议：


1. 毕业前练好写作

2. 毕业前学好 C 语言

3. 毕业前学好微观经济学

4.不要因为枯燥就不选休非计算机专业的课程

5.选修有大量编程实践的课程

6.别担心所有工作都被印度人抢走

7.找一份好的暑期实习工作

8. 寻找专业人士的帮助，培养你的自信心


我觉得写得都挺受用的。对于最后一条要强调一下，很多人似乎只做这一条，对前面几条根本不考虑。那么最后也是无济于事。你不可能指望别人说几句就让你功力大增，最重要的还是要你去做。

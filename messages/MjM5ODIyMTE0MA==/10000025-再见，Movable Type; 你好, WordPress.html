再见，Movable Type; 你好, WordPress

2013-01-03

总算有点空闲时间，把个人站点从 Movable Type 迁移到 WordPress 了。所以才迟迟没有更新小道消息。


微信上的各位朋友知道 Movable Type 和 WordPress 的估计不多。简单的说，这两个工具都是用来写 Blog 的。


坚持用 MT 好多年了，从 2004 年 12 月份开始启用，一直用到到现在，早期的 MT 程序优雅，静态发布功能深受好评，尤其是我这样的用户，当时站点托管在 Dreamhost 上，因为 CPU 不给力，每次 Rebuild 都要遇到几十次 500 才能通过，骂娘的心都有，但那个时候写东西真有热情。现在用 Linode，计算资源都快赶上几年前的标准服务器了，无需再担心 CPU 资源，但是写的东西反而少了，闲言碎语都通过微博说，也不需要记录下来，没什么实际价值。


之前说了好几回准备迁移，迟迟没动的原因是确实很不舍得，但 MT 现在真的不如 WP 方便，况且 WP 的插件众多，不断进步，我也不抱残守缺了，拥抱变化一回。


旧的站点内容都在，现在还会有少量 404 错误，我将尽快调整完毕。 再好的东西也会被超越，真是让人无可奈何。最近几年 MT 几乎没什么建树。只有日本那边开发社区还算活跃。中文社区连用户现在都屈指可数了。


通过 MT 间接认识了不少好朋友，回想那时候，真的挺好。


再见,Movable Type; 你好, WordPress.
